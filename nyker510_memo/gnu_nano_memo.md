# GUN nano editor

軽量テキストエディタ。MacOSとかUbuntuみたいなUnix系のOSによく搭載されている。編集方法が例によってわからなかったのよく使いそうなコマンドをまとめます。

command    | 意味
:--------- | :-----------
`Ctrl + O` | 現在のファイルを保存する
`Ctrl+G`   | ヘルプの表示
`Ctrl+A`   | 行頭へ移動
`Ctrl+K`   | 現在の行をカットする
`Ctrl+X`   | 現在のファイルをとじる
